# Config Package 개요

이 패키지는 모노레포 전체에서 공통으로 사용되는 개발 설정 및 의존성을 관리하는 곳입니다. 프로젝트 전반의 코드 일관성과 개발 환경 표준화를 담당합니다.

## 포함 내역

- **TypeScript 설정**: `tsconfig.base.json` 등 프로젝트 전반에서 확장하여 사용할 베이스 설정.
- **Linting & Formatting**:
  - **ESLint**: 공통 린트 규칙 및 플러그인 설정.
  - **Prettier**: 공통 코드 포맷팅 설정.
- **공통 개발 의존성**: 여러 패키지에서 공용으로 사용하는 빌드 도구, 테스트 유틸리티 등의 `devDependencies`.

## 사용 가이드라인

- **설정 공유**: 새로운 앱이나 패키지를 추가할 때 이 패키지에 정의된 설정을 `extends` 하여 사용하세요.
- **의존성 일원화**: 동일한 개발 도구가 여러 곳에서 서로 다른 버전으로 사용되지 않도록 이 패키지에서 버전을 관리하고 공유하는 것을 지향합니다.
- **일관성 유지**: 프로젝트 전체의 스타일이나 규칙을 변경해야 할 경우, 각 패키지를 수정하는 대신 이 패키지의 설정을 수정하여 일괄 적용되도록 합니다.
