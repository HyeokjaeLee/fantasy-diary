# Agent Server 개요

이 워크스페이스는 **Google Gemini**를 활용하여 소설을 생성하고 발행하는 핵심 에이전트 서버입니다.

## 역할 및 목적

- **소설 생성**: 사용자의 입력이나 테마를 바탕으로 Gemini AI 모델을 사용하여 소설의 시놉시스, 본문, 등장인물 등을 생성합니다.
- **콘텐츠 정제**: 생성된 텍스트를 서비스 규격에 맞게 가공하고 검수합니다.
- **발행 관리**: 완성된 소설을 데이터베이스(Supabase 등)에 저장하고 외부에 제공합니다.

## 인프라 및 배포

- **Runtime**: Bun
- **Cloud Platform**: Google Cloud Run
  - 서버리스 환경으로 배포되며, 트래픽에 따라 자동 스케일링됩니다.
  - 컨테이너 기반 배포를 위해 Dockerfile 및 Cloud Build 설정을 준수해야 합니다.
  - 모든 설정은 환경 변수를 통해 주입받으며, 상태 비저장(Stateless) 방식으로 설계합니다.

## 개발 가이드라인

- **성능 최적화**: AI 응답 대기 시간이 길어질 수 있으므로, 필요한 경우 비동기 처리 및 스트리밍 응답을 고려하세요.
- **에러 핸들링**: AI API 호출 실패, 할당량 초과(Quota Exceeded) 등에 대한 예외 처리를 철저히 하세요.
- **프롬프트 관리**: 소설의 퀄리티를 위해 프롬프트는 체계적으로 관리하고 필요시 별도의 설정 파일이나 DB에서 로드하도록 합니다.
